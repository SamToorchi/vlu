Abgerundete Ecken über CSS3 - Behavoristischer Ansatz
Theorie:
Sehr einfach können beliebigen Elementen über die CSS3-Anweisung border-radius, abgerundete Ecken mitgegeben werden. Zu den Elementen gehören, neben den üblichen HTML-Elementen wie DIV, genauso Bilder. Die Rundung kann individuell für jede „Ecke“ bestimmt werden.

Der einfachste Aufbau des Befehls, wenn alle 4 Ecken dieselbe Rundung bekommen sollen, lautet: 

border-radius: 20px;
Box mit normalen Ecken
Box mit abgerundeten Ecken

HTML:
<div class="rect_normal"></div>
<div class="rect_round"></div>


CSS:
.rect_normal{
 border: 2px black solid;
 width: 100px;
 height: 100px;
 margin: auto;
 margin-bottom: 10px;
 margin-top: 10px;
 background-color: #E8DCB3;
}

.rect_round{
 border: 2px black solid;
 width: 100px;
 height: 100px;
 margin: auto;
 margin-bottom: 10px;
 margin-top: 10px;
 background-color: #E8DCB3;
 border-radius: 20px;
}


Dadurch erhalten alle 4 Ecken eine Rundung von 20 Pixeln. Das klappt bei aktuellen Browsern, die CSS3 unterstützen.
Es können sowohl Pixelangaben, relative Angaben wie die Einheit em, wie auch prozentuale Angaben gemacht werden. Über eine prozentuale Angabe ist schnell die Erstellung eines Kreises (bzw. Ovals) möglich.

Kreis: 
border-radius: 50%;
Oval: 
border-radius: 150px / 100px;
Kreis
Oval

HTML:
<div class="circle"></div>
<div class="oval"></div>


CSS:
.circle{
 border: 2px black solid;
 width: 100px;
 height: 100px;
 background-color: #E8DCB3;
 border-radius: 50%;
}

.oval{
 border: 2px black solid;
 width: 100px;
 height: 100px;
 background-color: #E8DCB3;
 border-radius: 150px / 100px;
}


Praxis:
Aufgabe: Erstellen Sie einen roten Kreis in einem grünen Oval, wobei das Oval sich in einem abgerundeten blauen Rechteck befindet.
